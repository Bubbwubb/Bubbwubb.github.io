Description

I have recreated the hit game "Asteroids", if you have ever played it before you will know what it's about. If not, you controll a small ship and have to blast all the asteroids! Clearing all moves to the next level.

Outline

Controls

Using the arrow keys, you can steer the ship and activate the thrusters.
Pressing down on the spacebar will fire the blaster.

Scoring

You get points for every asteroid you blast, and then points for the smaller chunks that break off.

Graphics And Sound

This game was made using P5 and it's functionality. The ship and rocks were made using P5 to generate geometry. The sound was made using ToneJS and audio samples.

Requirements

In order to play this game you just need a computer with keyboard.
You will use the arrow keys to move.
Use the spacebar to shoot, and the enter button to restart from the game over screen.
